# Student-Placement-Data-Analytics

# Project Objectives
1. To collect and analyze student placement data to identify key factors influencing placement success.
2. To examine the relationship between students' academic performance, technical skills, internships, certifications, and placement outcomes.
3. To identify trends in placement rates across different departments, academic years, and companies.
4. To analyze salary packages offered to students based on their qualifications, skills, and academic achievements.
5. To create interactive dashboards using Power BI or Tableau for effective visualization of placement statistics and insights.
6. To perform exploratory data analysis (EDA) to uncover patterns, correlations, and anomalies in the placement dataset.
7. To provide data-driven recommendations that help students improve their employability and placement opportunities.
8. To assist training and placement officers in making informed decisions using analytical reports and visualizations.
9. To identify the most in-demand technical and soft skills required by recruiting companies.
10. To support educational institutions in enhancing placement strategies through data-driven insights.

# Process
The Student Placement Data Analytics project begins by defining the objective, which is to analyze student placement data and identify the key factors that influence placement outcomes. The first step is to collect student-related information, such as Student ID, Name, Gender, Department, CGPA, academic scores, internship experience, certifications, placement status, company name, and salary package, and store it in an Excel spreadsheet.

After collecting the data, the dataset is cleaned using Microsoft Excel by removing duplicate records, handling missing values, correcting formatting errors, standardizing department and company names, and validating numerical values such as CGPA and salary packages. Once the data is clean and consistent, Excel formulas, sorting, filtering, PivotTables, and charts are used to perform preliminary analysis and calculate key metrics, including total students, placed students, placement percentage, average CGPA, average salary package, and highest salary package.

The cleaned dataset is then imported into Power BI for advanced analysis and visualization. Using Power Query, the data is transformed by removing unnecessary columns, correcting data types, and preparing it for reporting. DAX measures are created to calculate important performance indicators such as total students, placed students, placement percentage, average salary, and average CGPA. These measures are used to build an interactive dashboard containing KPI cards, bar charts, column charts, pie charts, and slicers that allow users to analyze placement performance by department, gender, internship experience, certification status, company, and salary package.

Finally, the dashboard is used to generate meaningful insights and recommendations. The analysis helps identify departments with the highest placement rates, the impact of internships and certifications on employability, and the companies offering the highest salary packages. Based on these findings, recommendations are provided to improve placement performance, such as encouraging internships, promoting professional certifications, enhancing skill development programs, and strengthening relationships with recruiting companies. This complete process demonstrates how Excel and Power BI can be effectively used to transform raw student placement data into valuable insights that support informed decision-making.
